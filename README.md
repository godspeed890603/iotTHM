## 目的說明
在2016年，在使用了nano遭遇網路的問題，另一個機房改用Raspberry來開發來開發。</br></br>

## 架構說明
1.採用Raspberry透過SPI的通訊定控制液晶螢幕。</br></br>
2.裝設兩的DHT21溫溼度感應器</br></br>
3.提供NTP自動對時功能</br></br>
4.提供溫溼度上傳功能</br></br>
